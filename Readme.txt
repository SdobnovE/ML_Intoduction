# Описание файлов и решенных задач

1) log регрессия + линейная регрессия.ipynb - в файле реализован класс MySGDLinearModels. 

Класс может работать как с линейной регрессией, так и с логистической.

Выписана теория и функционал потерь, который нужно оптимизировать.

Для поиска коффициентов задачи используется градиентный спуск. От размера батча спуск может быть как просто градиентным, так и стохастическим

В ноутбуке произведено сравнение собственной реализации со стандартными sklearn-овскими. Время собственной реализации существенно выше, т.к. нужно оптимальнее выбирать параметры градиентного спуска.



2) knn регрессия + классификация.ipynb - в файле реализованы два класса MyKNeighborsClassifier и MyKNeighborsRegressor.

Отвечают они за метод k-ближайших соседей за классификацию и регрессию соответственно

Произведено сравнение точностей, которые совпадают для обоих классов. Это значит, что кастомная реализации верная
